# ロードマップ: Raw Image Viewer

## 1. 目的

本書は Raw Image Viewer の将来要件と拡張計画を管理する。
未実装機能の候補、期待される利用価値、段階的な実現方針を記載する。

現行実装の設計仕様は [基本設計](basic-design.md) と [詳細設計](detailed-design.md) を参照する。

## 2. 将来要件

### 2.1 インタラクティブ閲覧（Zoom & Pan）

- マウスホイールと UI ボタンで 1% から 1000% 以上のズーム
- 拡大時のドラッグパン
- 補間方法の切り替え
  - Nearest Neighbor（ピクセル確認向け）
  - Bilinear（視認性向け）

### 2.2 ピクセル情報インスペクション（Data Probe）

- カーソル座標（x, y）表示
- フォーマットに応じた生ピクセル値表示
  - RGB/BGR のチャンネル値
  - 16-bit 値
  - float 値
  - YUV 値

### 2.3 リアルタイム解析

- ヒストグラム（輝度またはチャンネル別）
- 統計表示（最小値、最大値、平均値、中央値）

### 2.4 動的 UI コントロール

- Webview 上のライブ調整（幅、高さ、フォーマット、ヘッダーサイズ）
- マルチバイトフォーマットのエンディアン即時切替
- float32/depth16 向けウィンドウ/レベルスライダー

## 3. 段階的な実装案

1. 操作性の改善

- ズーム、パン、補間切替を先行実装して表示体験を向上

2. 観測性の追加

- Data Probe を追加し、デバッグ用途に対応

3. 解析機能の追加

- ヒストグラムと統計情報を導入し、解析ユースケースに拡張

4. ライブ設定の統合

- UI でのパラメータ編集を可能にし、設定反復を高速化

## 4. スコープ外

本書は計画管理を目的とし、現行実装の正確な挙動定義は扱わない。
実装仕様は詳細設計で管理する。
